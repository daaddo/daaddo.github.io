Chi Sono

Sono un Backend Developer con una forte passione per la sicurezza applicativa, l'automazione e l'architettura software. Attualmente studio Programmazione e Gestione dei Sistemi Informatici all'Università di Perugia, dove approfondisco continuamente le mie competenze tecniche.

La mia esperienza si concentra principalmente su Java e il framework Spring Boot, con particolare attenzione alla progettazione di RESTful API sicure, all'implementazione di sistemi di autenticazione avanzati (JWT, OAuth2, Passkey) e all'integrazione di best practice come Clean Code e metodologie Agile.

Sono appassionato di DevOps e automazione: utilizzo Docker, GitHub Actions e strumenti di monitoring come Grafana, Prometheus e Loki per garantire performance ottimali e affidabilità dei sistemi. Mi piace creare soluzioni complete, dal database al deploy, con particolare attenzione alla scalabilità e alla manutenibilità del codice.

Socievole, pragmatico e sempre pronto ad apprendere nuove tecnologie, cerco costantemente di migliorare le mie competenze e di applicare soluzioni innovative ai problemi reali.
